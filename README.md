# Daily Timeline

A temporal note-taking and productivity system that captures your day as a chronological timeline. Designed with high-precision industrial minimalism.

## Features

- **Timeline**: Chronological feed of tasks, notes, and sessions.
- **Tasks**: Track actionable items with granular progress (0-100%).
- **Notes**: Freeform text entries.
- **Focus Timer**: Dedicated timer that logs sessions to your timeline.
- **Local First**: Data stored securely on your device (SQLite).

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the app**
   ```bash
   npx expo start
   ```

## Tech Stack

- Expo (React Native)
- SQLite
- TypeScript
